# tic-tac-toe-python
Simple terminal-based Tic Tac Toe game in python

This project was created to practice basic Python concepts such as:
- Conditional logic<br>
- Loops<br>
- Functions<br>
- List and indexing<br>
- Input and Output<br>

# how to play

   cd tic-tac-toe-python
